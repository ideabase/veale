== Description ==

This Plugin is designed to show your flickr photos into your WordPress site either as an album or as a gallery (A collection of albums).
There are plenty of flickr plugins out there. This one is a modern take and fully responsive and modern in design.

[youtube https://www.youtube.com/watch?v=SpcGlX_GMEs]

Basic version is limited to 50 images per album/gallery.
To get up to 500 images per album/gallery, get the pro version from https://www.srizon.com/srizon-flickr-gallery

== Installation ==

Search and install under plugins menu on your wp-admin.

If you want to download from zip then download the zip first and:

1. Go to your WordPress admin area and click on 'Plugins' menu.
2. Click 'Add New' and then click 'Upload Plugin'
3. Point the zip file and click 'Install Now'
4. Activate and find the newly added 'Flickr Gallery' Admin menu. Start there.
